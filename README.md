🚀 Getting Started

Zero Setup Required

This project is a single HTML file with all CSS and JavaScript embedded inline.

bash# Clone the repository
git clone https://github.com/yourusername/nayepankh-foundation.git

# Navigate into the folder
cd nayepankh-foundation

# Open directly in browser
open nayepankh_foundation.html
# or just double-click the file in your file explorer

No npm, no build step, no server needed. ✅
